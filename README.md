This app is a FoodMart app where users can see the available food items, and browse the items
Upon opening the app, the userʼs presented with a list of food items. 
The user can scroll to see all the available food items.
Each food item is shown with its image, price, name, and category.
If the food items fail to load, the userʼs shown an error message and a button to retry.

User can filter food items by categories, so that users can browse for items in specific categories.
The user can activate multiple category filters.
If no category filters are active, al food items are shown in the list. 
If any category filter is active, only food items in those categories are shown in the list.
